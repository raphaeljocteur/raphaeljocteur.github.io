  • Release : 2018
  • Language : C#
  • Engine : Unity

Description

At the bottom of the sea, some little fishes and other sea creatures are swimming to get rid off a naughty lantern fish who's chasing them.

What did i do ?

Procedural World Generation

We needed an infinite world made of gameplay brick randomly add gradually while progressing the level.

Basic Fish Gameplay

Implementing a basic gameplay like flappy bird but with a different gravity (water).

Gameply Mod

Implementing defeat condition and boss animation. Each hit on player make the lantern fish come closer, if you don''t get hit for a while youu will gain back some distance.

Basic Interactibles Elements

Implementing cuttables seaweed (sweep screen), and enemies fishes.
